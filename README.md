# Animated Card

[Live version](https://animated-card-scss.netlify.app/)

### To install

```
npm -g sass
npm -g http-server
```

### Building

In 2 terminal windows:

```
# Terminal Window 1
sass --watch scss/main.scss:css/main.css

# Terminal Window 2
http-server .
```

### Running

When both commands are running, open `http://localhost:8080` in your web browser.
